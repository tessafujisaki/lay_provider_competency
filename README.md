# Non-specialist provider- and facility-level correlates of competence in perinatal mental health skills in Western Kenya: a cross-sectional study

## Overview
The de-dentified R code for the manuscript "Non-specialist provider- and facility-level correlates of competence in perinatal mental health skills in Western Kenya: a cross-sectional study" [currently under review at PLOS Mental Health] is included in this Github repository.

Basic description of the code:

#### General

| Variable       | Description                                            |
| -------------- | ------------------------------------------------------ |
| `pt_id`     | Anonymized participant ID (numeric)                       |

#### Provider Demographic Characteristics

| Variable       | Description                                                                          |
| -------------- | --------------------------------------------------------------------------           |
| `cal_age`           | Calculated age at enrollment                                                    |
| `age_category`      | Age category (<30, 30-40, >40)                                                  |
| `education`         | Education level (diploma, primary, secondary, bachelor's degree or higher )     |
| `education_category`| Education category (high school diploma or lower, bachelor's degree or higher)  |
| `cadrel`            | Cadre (nurse, clinical officer, HIV testing counselor, mentor mother)           |
| `sex`               | Sex (female or male)                                                            |
| `yearsjob`          | Total years working in job                                                      |
| `yearswork`         | Total years working in current facility                                         |

#### Stigma and Implementation Outcome Assessment Measures

| Variable              | Description                                                                    |
| --------------------- | --------------------------------------------                                   |
| `stigma_mean`         | Mean Opening Minds Stigma Scale for Health Providers (OMS-HC) score            |
| `oric_mean`           | Mean Organizational Readiness for Change (ORIC) score                          |
| `aim_mean`            | Mean Acceptability of Intervention Measure (AIM) score                         |
| `iam_mean`            | Mean Intervention Appropriateness Measure (IAM) score                          |
| `fim_mean`            | Mean Feasibility of Intervention Measure (FIM) score                           |
| `aaf_mean`            | Mean AIM, IAM, FIM Accessibility, Appropriateness, and Feasibility (AAF) score |


#### Facility-Level Charcteristics 
| Variable              | Description                                                                    |
| --------------------- | --------------------------------------------                                   |
| `Facility`            | Facility name                                   |
| `Level`               | SAMHSA/HRSA Behavioral Health Integration Level |
| `new_anc_volume`      | Monthly new antenatal care attendees            |

#### Outcomes

| Variable                | Description                                                 |
| ----------------------- | -----------------------------------------                   |         
| `equip_total`           | EQUIP (Ensuring Quality in Psychological Support) sum score |
